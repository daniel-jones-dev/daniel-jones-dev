## Hi, I'm Daniel 👋

Backend engineer at [DeepL](https://www.deepl.com).
I like building things developers actually use - APIs, SDKs,
the stuff where the interface matters as much as what's underneath. I care a lot
about backward compatibility (I know how much a breaking change costs the people
on the other end) and code that explains itself.

I pick up new programming languages for fun - almost every one has an
interesting feature. I use AI tools heavily everyday, Claude Code in particular.

Most of my public work lives at [@DeepLcom](https://github.com/DeepLcom).
